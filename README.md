# Основы PHP (семинары в записи)
## Урок 9. Улучшаем приложение
Скорректируйте список пользователей так, чтобы все пользователи с правами администратора в таблице видели две дополнительные ссылки – редактирование и удаление пользователя. <br />
При этом редактирование будет переходить на форму, а удаление в асинхронном режиме будет удалять пользователя как из таблицы, так и из БД.
